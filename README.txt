== Description ==

Allow user to vote on your article and gather feedback on how helpful your articles are.

== Installation ==

1. Use the WordPress plugin installer to install the plugin. Alternatively, you can just extract the folder in our download package and upload it to your plugin directory.

2. Access the Plugins admin menu to activate the Article Voting plugin.
